## Migration Notes

### Upgrading from 1.x to 2.x
1. Clear browser cache or wait for service worker update
2. Re-enter API key if settings reset
3. Bookmarks preserved in LocalStorage

### Cache Reset
If experiencing issues:
1. Open Settings
2. Click "Reset Cache" button
3. Reload the app
